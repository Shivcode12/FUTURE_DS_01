# FUTURE_DS_01
# College Event Feedback Analysis (Power BI)

## Project Overview

This repository contains a **Power BI dashboard** created to analyze **college event feedback** collected from participants. The goal is to convert raw feedback data into meaningful insights that help organizers understand participant satisfaction and improve future events.

The Power BI file provided (`Task 1.pbix`) focuses on:

* Ratings analysis
* Participant feedback trends
* Overall event performance
